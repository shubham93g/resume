## Fonts
IBM Plex Sans is self-hosted as TTF (`fonts/`) rather than loaded from Google Fonts. Chrome's PDF engine embeds WOFF2 with incomplete character mapping tables, which breaks text selection in PDF viewers. TTF preserves the full mapping.
